## Quick install

Wealtify-js is constantly in development! Try it out now:

### NPM

```sh
npm i wealtify-js
```

**or**

### Yarn

```sh
yarn add wealtify-js
```

### Import

After installation, you can import our functions  your project using this snippet:

```sh
@import * from 'wealtify-js'
```
